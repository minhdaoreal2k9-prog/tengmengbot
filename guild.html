How to Use the Bot (New User Guide)

Welcome! Here are the simple steps to get started with the bot:

Step 1: Send Your Email and Code

First, you need to provide the Email (linked to your game account) and the Login Code (verification code) to the admin.

Step 2: Admin Assigns Permission

After receiving your info, the admin will assign bot permissions to your account. When successful, the bot will send a confirmation message.

Step 3: Start the Bot (Using `start`)

To begin, simply type the start command. The bot will activate and report its status.

Step 4: View the Screen (Using `show`)

After the bot has started, you can type the show command at any time. The bot will take a screenshot of the game and send it to you.

Step 5: Stop the Bot (Using `stop`)

If you want to pause the bot, just type the stop command. When you're ready to run it again, simply type start.
